# sales-trend-analysis
# Sales Trend Analysis 2023

## 📊 Project Overview
Analysis of sales data to identify revenue trends across months, quarters, categories and regions.

## 🔍 Business Questions Answered
- How does revenue trend month by month?
- Which quarter performs best?
- Which category drives the most revenue?
- Which region contributes the most?

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## 💡 Key Findings
- December was the strongest month ($17,906)
- Q4 generated 38% of total annual revenue
- The business grew 219% from January to December

## 📁 Files
- `sales_trend_analysis.ipynb` - Main analysis notebook
- `sales_trends.csv` - Dataset used
